# Dashboard de Qualidade dos Concursos

Este mini-site interpreta o arquivo:

```text
data/concursos/_relatorio-qualidade.json
```

## URLs no GitHub Pages

Depois de subir o patch, acesse uma destas opções:

```text
/dashboard-qualidade.html
```

ou:

```text
/dashboard-qualidade/
```

## Como usar

1. Rode o workflow **Relatório de Qualidade - Concursos**.
2. Aguarde o commit do relatório.
3. Abra o dashboard.
4. Veja as filas: rodar qualificado, revisar, fonte falhou e OK.
5. Copie o workflow sugerido das instituições prioritárias.

## Uso manual

Também é possível colar o JSON diretamente no campo do painel ou carregar um arquivo `.json` pelo seletor de arquivo.
